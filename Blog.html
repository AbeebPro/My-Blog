Is HTML a programming language?

Well that's a question I've found most beginners in web development asking.Html is not a programming language but a mark-up language every web developer should know. It is a language that gives structure to whatever website/webapps you want to build. With consistent practice, it's a very simple language you can learn in little or no time. There are many concepts in HTML but you don't have to try memorizing them all. It's a language that evolves every now and then. Perhaps, you've heard of HTML4? Probably. The most recent version of html right now is HTML5 with really interesting new concepts.

Need Help?

HTML, generally, has a wide array of elements and attributes, so if you find yourself needing help, you can visit sites like w3schools, or MDN. Those are the top two sites with resources and documentation on HTML and many programming languages you'll be needing in your web development journey.
